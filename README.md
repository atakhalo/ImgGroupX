# 核心功能
分组网格视图：
1. 以文件夹作为分组；或可手动创建临时分组；
2. 每组以网格视图组织；
3. 多组集合查看
4. 支持彩虹模式（按层级着色）
# 其他功能
对比功能
1. 可选择两张图片进行对比

筛选功能
1. 可按文件名、分组、路径 进行筛选；
2. 可保存为预设

性能相关
1. 支持异步扫描、异步加载、懒加载
