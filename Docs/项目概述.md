# ImgGroup - 图片分组浏览器

## 项目简介

ImgGroup 是一个基于 **Tauri v2 + Vue 3 + Rust** 的桌面端图片分组浏览器。支持多文件夹管理、目录树分组、大图查看（分组内导航）、图片对比、三模式筛选排序、虚拟分组（保存到文件夹/替换分组）、批量移动/复制（保留目录结构、同名自动重命名）、文件夹回收站删除、多层级多选操作、全选/反选/一级图片快速选择、自定义外部打开方式、分组标题颜色定制、多语言（中/英）、命令行参数、配置持久化等完整图片浏览功能。

## 技术栈

| 层 | 技术 |
|---|------|
| 前端框架 | Vue 3 (Composition API + `<script setup>`) |
| 构建工具 | Vite 6 |
| 类型检查 | TypeScript 5 + vue-tsc |
| 桌面框架 | Tauri v2 |
| Rust 后端 | Tauri Commands + walkdir + imagesize + kamadak-exif + serde_yaml + notify |
| 系统对话框 | @tauri-apps/plugin-dialog |
| 文件打开 | @tauri-apps/plugin-opener |

## 项目结构

```
ImgGroup/
├── index.html                    # 入口 HTML
├── package.json                  # 前端依赖
├── vite.config.ts                # Vite 配置
├── tsconfig.json                 # TypeScript 配置
├── Docs/                         # 项目文档
│   ├── 项目概述.md
│   ├── 功能清单.md
│   ├── 技术架构.md
│   └── 开发指南.md
├── src/                          # 前端源码
│   ├── main.ts                   # Vue 入口
│   ├── App.vue                   # 主应用组件（拖入、滚轮缩放、配置管理、文件变更监听）
│   ├── types/index.ts            # 类型定义（ImageItem, FolderNode, AppSettings 等）
│   ├── stores/imageStore.ts      # 全局状态 & API（树构建、筛选、配置持久化、路径排除、文件监听、刷新）
│   └── components/
│       ├── GridView.vue          # 网格视图（按需加载、排序筛选、可覆盖 bgColor）
│       ├── GridItem.vue          # 网格卡片（固定高度+宽高比自适应）
│       ├── FolderPanel.vue       # 文件夹面板（展开管理、筛选联动）
│       ├── FolderGroup.vue       # 文件夹组（递归、居中标题、彩虹着色、box-shadow 边框）
│       ├── ImageViewer.vue       # 大图查看器（分组内导航、外部程序打开）
│       ├── CompareViewer.vue      # 对比视图（两图重叠、分割线、缩放平移）
│       ├── OperationBar.vue      # 大图操作栏（缩放、全屏、打开菜单）
│       ├── ControlBar.vue        # 底部控制栏（模式切换、清空、新分组、彩虹开关、刷新）
│       ├── FilterSortBar.vue     # 筛选排序栏（三模式筛选、排序、右侧插槽承接加载面板）
│       └── SettingsPanel.vue     # 设置面板（样式、颜色、预设、外部程序、彩虹颜色配置）
└── src-tauri/                    # Rust 后端
    ├── Cargo.toml                # Rust 依赖（含 serde_yaml）
    ├── tauri.conf.json           # Tauri 配置（dragDropEnabled）
    ├── capabilities/default.json # 权限配置
    └── src/
        ├── main.rs               # 程序入口
        └── lib.rs                # Tauri Commands（扫描、base64、EXIF、外部打开、配置）
```

## 启动方式

```bash
cd ImgGroup
npm run tauri dev    # 开发模式
npm run tauri build  # 生产构建
```
