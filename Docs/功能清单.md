# 功能清单

## 图片加载

| 功能 | 说明 | 状态 |
|------|------|------|
| 打开文件夹 | 原生文件夹选择器，递归扫描所有图片 | ✅ |
| 打开图片 | 文件选择器，选中图片自动创建虚拟分组 | ✅ |
| 拖入文件夹 | 识别为文件夹，添加到目录树 | ✅ |
| 拖入图片 | 识别为文件，自动创建虚拟分组 | ✅ |
| 混合拖入 | 同时拖入文件和文件夹，分别处理 | ✅ |
| 图片去重 | 按路径去重，重复加载不重复显示 | ✅ |
| 支持格式 | jpg/jpeg、png、webp、bmp、gif | ✅ |
| 渐进式扫描 | 后台线程遍历，每扫描完一个目录立即渲染，无需等待全量完成 | ✅ |
| 扫描状态指示 | 顶部加载栏显示旋转 spinner + "扫描中"，不阻挡操作 | ✅ |

## 视图模式

### 紧凑模式（分组关闭）
- 所有图片在同一网格中平铺显示
- 虚拟分组作为独立区域插入

### 目录分组模式（分组开启）
- 树状结构展示文件夹层级
- 根节点 = 加载的文件夹（显示父路径 + 文件夹名）
- 子节点 = 子文件夹（居中显示文件夹名）
- 叶子节点 = 图片网格
- 拖入图片自动创建虚拟分组（📦 节点）
